# DM Invest SA - Site Vitrine

Site vitrine moderne pour DM Invest SA, gestionnaire de fortune suisse.

## Stack technique

- **Next.js 14** avec App Router
- **React 18**
- **TypeScript**
- **Tailwind CSS**

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Build de production

```bash
npm run build
npm start
```

## Design

Le site utilise une direction artistique "old money suisse" avec :
- Palette de couleurs : vert forêt (#12372A), ivoire (#F5F1E8), anthracite (#2C2C2C), doré mat (#C9A961)
- Typographie : Playfair Display (titres) et Inter (texte)
- Design sobre, élégant et institutionnel
- Responsive et accessible
